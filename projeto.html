O Projeto
A ideia de construir um site para treinar inglês – com foco especial em números – surgiu quando Marlon, ao estudar os números básicos, não encontrou nenhum site que oferecesse exercícios ativos para praticar e fixar o conteúdo de forma prática.
O Projeto
Visualizar

Dessa necessidade, nasceu a versão 1.01, desenvolvida integralmente com o Chat GPT(visual e lógica), permitindo de forma simples treinar sobre os números.

Cinco meses depois, surgiu a iniciativa de criar uma nova versão. Com novos conteúdos, funcionalidades e visual aprimorado. A nova versão (V2.01), teve o design, e HTML e CSS feito por Marlon. Enquanto a lógica (JavaScript), foi estruturado com o Chat GPT.

Além de diversas correções, testes e ajustes realizados por Marlon para garantir que tudo funcione com fluidez.
